Simple E-Commerce Website
This is a basic e-commerce website template with an interactive navigation menu. The navigation menu changes color when scrolled and when hovering over a menu item. The website includes sections for Home, Shop, About, and Contact.

Features
Fixed navigation menu that remains visible on all pages.
Menu changes background color when the page is scrolled.
Menu items change color when hovered over.
Simple, clean layout with sections for Home, Shop, About, and Contact.

Technologies Used
HTML
CSS
JavaScript

Prerequisites
You need a web browser to view the website. No additional software or libraries are required.

Customization
You can customize the website by editing the HTML, CSS, and JavaScript files:

HTML: Modify the content and structure of the website.
CSS: Change the styles, colors, and layout.
JavaScript: Add more interactivity or modify existing behavior.
